# [サービス名]

カラオケお助けアプリ
SongooD!

## サービス概要

カラオケで「あの曲歌いたかったのに思い出せなかった...」「」
本アプリは、歌いたい曲と自分に合うキーを事前に登録しておくことで、
カラオケで迷わず・失敗せず、最高の時間を過ごせるお助けアプリです。

## 開発背景

カラオケボックスから出た後に「あの曲歌いたかっただが曲名思い出せなかった、後悔、、」
曲を入れたはいいものの「このキーだとなんだか歌えない」
そういった、痒いところに手が届くアプリを作ることで
最高のカラオケタイムを送ることができる、そう思い開発いたしました。

## ターゲット層

ターゲット歌うこと、カラオケが大好きな１０ − ３０代
一人でもよく行くぐらい大好きである、うまいとは思っていない。
いつも原曲キーで歌うため難しい曲は基本的に諦めている
カラオケに行くと周りの年齢層を意識した曲選びをするがいつも迷ってしまう
カラオケのキーが分からず、歌いたい曲を歌えない

### ターゲット層の理由

自分をペルソナにしてみました。
自分以上にカラオケに行く方があまりいなかったため。
その方が考えやすいと思いました。

### ユーザー獲得方法

１：認知してもらうために実際に使用した１５秒程度のショートムービーを作成し、拡散
（YOUTUBE ショート、TIKTOK、インスタショート）
２：カラオケに行く RUNTEQ 生の方、友人周りから使用していただき、レビューを SNS 等に投稿
フィードバック改善を行う

## サービス利用のイメージ

・カラオケで使うことを考慮し、スマホ前提の開発
・あらかじめ入れておいたプレイリストを見ることで
急に歌を振られても焦らず選曲できる

好きな曲を歌えるようにしたい思いを叶えられる。
キーを設定して保存する機能もある

### サービス差別化のポイント・推しポイント

設定したタグによって好きな気分などでも検索できる。

全体的に質素な UIUX が多く、機能性はあっても楽しさを感じないので
リッチな UIUX にしユーザーの満足感を向上させる

### MVP で実装予定の機能

- ユーザー登録・ログイン機能(Devise 使用)
  曲の登録機能
  - 曲名、アーティスト名
  - 自分に合うキー(±0〜6 の選択式)
  - メモ欄(歌い出しの歌詞など)
  - ジャンル・気分タグ(複数選択可)
  - 歌った後の感想を入れられる（難しい、歌えた、ちょいとキツイなど）
  - 自分の歌を点数化（スライドバー）

### 管理機能

- マイプレイリスト一覧表示
  - 登録した曲を一覧で確認
  - カード形式で見やすく表示
- 曲の編集・削除
- 検索・フィルタ機能
  - アーティスト名で検索
  - タグで絞り込み(盛り上がる曲、しっとり系など)

## 本リリース以降で実装予定の機能

- Spotify API 連携
  - 曲のプレビュー再生機能
  - アルバムジャケット表示
- プレイリスト共有機能
  - 友達とプレイリストをシェア
- 年代別おすすめ曲表示
  - 80 年代、90 年代、2000 年代などの人気曲を提案
- 音域分析機能
  - 自分の音域を登録して、歌いやすい曲を提案
  - 最適なキーの提案

## 使用する技術スタック

## 高度な技術要素

### 1. Hotwire によるリッチなインタラクション

- Turbo Frames でページ遷移なしの編集
  - 曲の編集時、ページ全体をリロードせずに編集フォームを表示
- Stimulus でタグの動的追加・削除
  - ユーザーが自由にタグを追加・削除できる

### 2. 検索・フィルタ機能の実装

- Ransack を使った複数条件検索
  - 曲名、アーティスト名、タグで絞り込み
- タグの複数選択フィルタ
  - 「盛り上がる曲」×「2000 年代」のような組み合わせ検索

### 3. レスポンシブデザインと UI/UX

- Tailwind CSS でモバイルファーストの UI
- カラオケ店でも見やすい大きめのボタン・文字
- 直感的に操作できるシンプルな画面設計
